## 一、CRITICAL / 必须立即修复 (5项)

### BUG-1: `bindZeroInputEvents()` 函数不存在导致搜索崩溃
- **来源**: 代码质量审计
- **位置**: `js/app.js` 第 2535 行
- **影响**: 用户清空搜索框时触发 `ReferenceError`，搜索功能完全崩溃
- **修复**: 删除该调用行（事件已由委托监听器处理）

### BUG-2: `navigate('path', {pathId:'...'})` 传入对象导致路径跳转失败
- **来源**: 代码质量审计
- **位置**: `js/app.js` 第 1120、1168 行
- **影响**: "推荐路径"按钮点击后跳转到 `#path/[object Object]`
- **修复**: 改为 `navigate('path', lp.id)`

### SEC-1: Content-Security-Policy 完全缺失
- **来源**: 安全审计
- **位置**: `vercel.json`
- **影响**: 无CSP保护，若DOM XSS被触发可加载任意外部脚本
- **修复**: 添加 CSP + HSTS 响应头

### SEC-2: 教程内容 `s.body` 直接 innerHTML 渲染（DOM XSS）
- **来源**: 安全审计
- **位置**: `js/app.js` 第 1668 行及多处
- **影响**: JSON数据如被篡改，恶意HTML直接执行
- **修复**: 引入 DOMPurify 白名单清洗

### A11Y-1: 全站键盘不可访问
- **来源**: 无障碍审计
- **位置**: `js/app.js` 约15处模板字符串
- **影响**: 大量 `<div onclick>` 交互元素无 `tabindex`，键盘用户完全无法操作
- **修复**: 改为 `<button>` 或添加 `tabindex="0"` + `role="button"` + keydown

## 三、MEDIUM / 中优先级 (20项)

### 安全 (4项)
- 用户邮箱明文存储 localStorage 且写入 analytics
- analytics日志记录敏感数据无清理机制
- hash路由参数未白名单验证
- Google Fonts无SRI（固有限制）

### 性能 (5项)
- innerHTML大规模DOM重建（26处），筛选/翻页每次全量替换
- CSS动画和GPU层过多（11动画+7 backdrop-filter）
- 全局搜索每次全量遍历8个数据集
- News日期排序重复执行 `new Date()` 解析
- scroll监听器未合并（2个独立监听）

### 无障碍 (5项)
- `--text-muted (#4e5a6e)` 对比度仅3.0:1（需4.5:1）
- 缺少 `prefers-reduced-motion` 支持
- 导航/筛选按钮缺少ARIA状态标记
- 搜索覆盖层非正确dialog
- 搜索结果数量变化不通知

### 用户体验 (3项)
- 首屏数据加载期间无状态反馈（白屏）
- 工具/模型/Prompt/资源 四个页面无行内搜索
- 教程118篇无分页

### CSS架构 (3项)
- 5824行单文件（需拆分为15-20个模块）
- 未定义变量 `--card-bg` 被引用17次
- 259处 rgba() 硬编码绕过Token系统
